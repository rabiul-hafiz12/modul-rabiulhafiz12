# Modul 13 – Git dan GitHub untuk Kontrol Versi

Proyek ini digunakan untuk latihan dasar kontrol versi menggunakan Git dan GitHub
sebagai persiapan mini project website interaktif pada Pemrograman Web 1.

## Deskripsi

Halaman web sederhana yang menampilkan ringkasan perintah Git, alur kerja Git,
dan identitas mahasiswa. Dikembangkan sebagai bagian dari praktikum Modul 13.

## Fitur

- Tampilan ringkasan perintah Git dalam format tabel interaktif
- Visualisasi alur kerja Git (Edit → Add → Commit → Push)
- Tombol ubah deskripsi menggunakan JavaScript
- Desain bertema merah konsisten dengan laporan praktikum

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (DOM & Event)
- Git & GitHub (kontrol versi)
