Ringkasan Perintah Git

Perintah Fungsi
git initMembuat repository lokal
git statusMelihat status perubahan file
git add .Menambahkan file ke staging area
git commit -mMenyimpan perubahan ke riwayat Git
git log --onelineMelihat riwayat commit singkat
git pushMengirim commit ke GitHub
git pullMengambil perubahan dari GitHub
git branchMembuat atau melihat daftar branch
git mergeMenggabungkan branch
git cloneMenyalin repository dari GitHub
